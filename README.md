## Portfolio Build and Run

This portfolio is built with Angular and focuses on a pixel-art-inspired UI.

### Development server

```bash
ng serve
```

Then open `http://localhost:4200/`. The app reloads on file changes.

### Build

```bash
ng build
```

Build output is in `dist/`.

### Tests

```bash
ng test
```
